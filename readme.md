# プロダクト名

To Do アプリ

## 課題内容（どんな作品か）

- タスクを入力し、追加ボタンを押すと、下のTo Doにリスト表示される
- 追加されたタスクは完了ボタンも合わせて表示される
- 完了ボタンを押すと、localStorageと画面から消える
- 全てのタスクが消えると"Complete! Well Done!"のメッセージが表示される
- タスクの増減はストレージにも反映される
